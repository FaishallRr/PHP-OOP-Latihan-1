# Praktikum Pemrograman Web Lanjut - OOP PHP

Repository ini berisi hasil latihan praktikum mengenai **Object Oriented Programming (OOP) menggunakan PHP**.

## Deskripsi

Pada praktikum ini dilakukan pembuatan class `Mahasiswa` yang memiliki beberapa property dan method untuk menyimpan serta menampilkan data mahasiswa. Program dijalankan menggunakan PHP melalui web browser dengan server lokal.

## Tugas Praktikum

1. Kumpulkan semua latihan yang dilakukan, termasuk yang ada di materi PPT ke GitHub, kemudian kirimkan link repository ke Kulino.
2. Kumpulkan screenshot kode program dan hasil percobaan dari browser (lengkap dengan URL yang diakses) dalam bentuk dokumen PDF, kemudian upload ke Kulino.
